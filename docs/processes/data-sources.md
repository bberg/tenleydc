# Data Sources Guide

> Where to find information for neighborhood history content and how to use each source.

---

## Primary Archives

### Library of Congress

**URL:** https://www.loc.gov/

**Key Collections:**
- **Theodor Horydczak Collection** (~14,350 photographs of DC, 1920s-1950s)
- **FSA/OWI Photographs** (Depression-era documentation)
- **Civil War Photographs** (Brady, Gardner collections)
- **Maps Division** (historical maps, Sanborn fire insurance maps)
- **Chronicling America** (digitized historical newspapers)

**How to search:**
1. Use LOC.gov search with neighborhood name + date range
2. Browse specific collections by geographic area
3. Download high-res images (public domain)

**Citation format:**
> Photo by [photographer name], [date]. Library of Congress, [collection name].

### Chronicling America (Newspapers)

**URL:** https://chroniclingamerica.loc.gov/

**Coverage:** 1789-1963 (varies by paper)

**Key DC Papers:**
- Evening Star (1852-1981) - Primary DC coverage
- Washington Post (1877-present) - Major events
- Washington Times (various periods)

**Search tips:**
- Use exact neighborhood name in quotes
- Filter by date range and state (DC)
- Export citations for documentation

**Example findings:** 15,692 articles mentioning "Tenleytown" available

### DC History Center

**URL:** https://dchistory.org/

**Catalog:** https://catalog.dchistory.org/

**Key holdings:**
- Photographs of DC neighborhoods
- Maps and atlases (Hopkins Atlas, etc.)
- Vertical files by topic/neighborhood
- Oral history collections

**Access:** Some items require in-person visit; many digitized

**Example items:**
- CHS 04094 (Fort Pennsylvania sketch)
- CHS 07317 (1887 Hopkins Atlas)
- CHS 06914 series (Immaculata Seminary)

### DC Public Library - People's Archive

**URL:** https://dclibrary.org/research

**Key resources:**
- Washingtoniana Collection
- Neighborhood vertical files
- Historical photograph collection
- City directories
- Building permits

**Branch resources:**
- Martin Luther King Jr. Library (main research)
- Local branch special collections

### National Archives

**URL:** https://www.archives.gov/

**Key records:**
- Census records (detailed demographics)
- Military records (fort construction, personnel)
- Land records and patents
- Federal agency records

### Internet Archive

**URL:** https://archive.org/

**Key materials:**
- Digitized books and publications
- Audio/video recordings
- Wayback Machine (historical websites)
- Community-uploaded materials

**Example:** 31 Fort Reno items (punk/hardcore concert history)

**URL:** https://planning.dc.gov/HPO

- Building permits

- Census tract data
- School boundaries

### National Park Service

**URL:** https://www.nps.gov/

- Civil War defenses documentation
- National Register listings
- Historic American Buildings Survey (HABS)

## Research Strategies

### For Neighborhood History

1. **Start with landmark nominations** - Most comprehensive existing research
2. **Search Chronicling America** - Date-range newspaper coverage
3. **Check LOC photo collections** - Visual documentation
4. **Review census records** - Demographic data by decade
5. **Contact historical society** - Local expertise and collections

### For Building/Business History

1. **Building permit records** (DC Open Data)
2. **City directories** (DC Public Library)
3. **Sanborn maps** (LOC, DC History Center)
4. **Newspaper archives** (opening/closing announcements)
5. **Tax records** (DC government)

### For Demographic History

1. **Census records** (National Archives, Ancestry)
2. **DC Office of Planning** (current demographics)
3. **Academic studies** (JSTOR, university libraries)
4. **Newspaper coverage** (integration, community change)

### For Transportation History

1. **DC Transit Heritage** (enthusiast documentation)
2. **WMATA archives** (Metro construction)
3. **Streetcar company records** (DC History Center)
4. **Newspaper coverage** (route announcements, service changes)

## Data Collection Workflow

### Phase 1: Digital Archive Sweep

```
For each neighborhood:
1. LOC search: "[Neighborhood] Washington DC"
   - Filter by photographs, maps, newspapers
   - Download relevant high-res images

2. Chronicling America search: "[Neighborhood]"
   - Filter by DC newspapers, date range
   - Document article counts by era
   - Save key article citations

3. DC History Center catalog search
   - Note relevant item numbers
   - Plan in-person visit if needed

4. Internet Archive search
   - Check for digitized local publications
   - Search for audio/video materials
```

### Phase 2: Government Sources

```
For each neighborhood:
1. Historic Preservation Office
   - Download all landmark nominations in area
   - Review National Register listings

2. DC Open Data
   - Export building permits by area
   - Download census tract demographics

3. DC Office of Planning
   - Review comprehensive plan chapters
   - Get neighborhood profile data
```

### Phase 3: Local Sources

```
For each neighborhood:
1. Identify historical society
   - Research existing publications
   - Note contact for outreach

2. Identify Main Street organization
   - Review business directory
   - Note event calendar sources

3. Identify local library branch
   - Check for special collections
   - Note vertical file availability
```

## AI-Generated Content Policy

When using AI-generated images:

1. **Always disclose:** "*AI-generated artistic interpretation of [subject]. Not a historical photograph.*"
2. **Use only when:** Archival images don't exist for important scenes
3. **Never present as:** Historical documentation
4. **Label clearly:** In both caption and alt text

## Rights and Attribution

### Public Domain Sources
- Library of Congress (most pre-1928 materials)
- Federal government publications
- Expired copyrights

### Attribution Required
- All photographs: Credit photographer and collection
- Maps: Credit creator and repository
- Newspaper excerpts: Citation with date and page

### Permission Required
- Living photographers' works
- Recent publications
- Private collections

### Fair Use Considerations
- Short quotes for educational purposes: Generally OK
- Full article reproduction: Not OK
- Image thumbnails with links: Generally OK
- High-res commercial use: Not OK without permission
