## Cost Estimate

- **16 domains × ~$12/year** = ~$192/year
- First year with typical registrar: ~$150-200 total
- Renewals: ~$192/year

---

## Registrar Options

| Registrar | Typical .com Price | Notes |
|-----------|-------------------|-------|
| Cloudflare | $9.15/year | At cost, best value, easy DNS |
| Namecheap | $10-13/year | Good value, frequent sales |
| Google Domains | $12/year | Simple, integrates with Google |
| GoDaddy | $12-20/year | Avoid - upsells, higher renewal |

**Recommendation:** Cloudflare Registrar - cheapest, and DNS already configured

---

## Domain Configuration Checklist

After purchase, for each domain:

- [ ] Add to Cloudflare
- [ ] Configure DNS records:
  - `A` record → Railway IP
  - `CNAME` for `www` → main domain
- [ ] Enable SSL (automatic with Cloudflare)
- [ ] Configure Railway custom domain
- [ ] Verify HTTPS working
